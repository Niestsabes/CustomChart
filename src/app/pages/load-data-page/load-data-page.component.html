Instructions

Pour pouvoir analyser vos données, notre solution d'analyse en ligne gratuite requiert uniquement les choses suivantes :

  • Format : Un fichier .csv.
  • Variables/Individus : Les variables doivent être en colonne, et les individus en ligne.
  • Taille : Pas de restriction sur la taille du jeu de données.

Chargez votre jeu de données

Pour débuter l'analyse de vos données, il vous suffit simplement de charger votre fichier de données et de choisir le séparateur puis d'appuyer sur Lancer l'analyse. Assurez-vous que le format de votre fichier est conforme aux instructions.
